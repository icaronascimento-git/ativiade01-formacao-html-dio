Anotações

  • - Método GET para envio do form, os dados de envio do formulário aparecerão na URL.
  • - No método POST, os dados não aparece na barra de URL quando o form for enviado para o servidor
  • - Podemos usar o atributo target no form, com ele poderemos indicar se queremos que os dados do form trafeguem mesma página ou se abre uma outra página para trafegar os dados e manter a página do form intacta.
  • - O atributo autocomplete ON/OFF do form indica se desejamos que ao acessar a página novamente, os dados quer inserimos nos campos do form em outro momento, sejam "lembrados" e o navegador sugira o auto preenchimento.
  • - Os atríbutos iniciados com "ON" são chamados de eventos, usados em conjunto com o JavaScript
